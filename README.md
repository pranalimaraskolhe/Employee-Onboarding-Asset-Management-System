# Employee-Onboarding-Asset-Management-System
The project automates the entire employee onboarding lifecycle inside Salesforce.

# Business Problem:
When a new employee joins an IT company, the onboarding process involves multiple manual steps: creating system accounts, assigning a laptop and peripherals, setting up access, scheduling orientation sessions, and notifying multiple teams. Without automation, this process takes 2-3 days and involves constant follow-up emails. This project automates the entire employee onboarding lifecycle inside Salesforce.
